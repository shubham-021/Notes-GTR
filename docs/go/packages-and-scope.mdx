## Package Level Variables

When a variable is used by many functions, it doesn't make sense to pass it from function to function. Define it as a package level variable.

Package level variables:
- Defined at the top, outside all functions
- Must be declared using `var` (cannot use `:=`)

```go
package main

const conferenceTickets = 50
var conferenceName = "Go conference"
var remainingTickets uint = 50

func main() {
    greetUser()
}

func greetUser() {
    fmt.Printf("Welcome to %v booking application\n", conferenceName)
    fmt.Printf("We have total of %v tickets and %v are still available.\n", 
        conferenceTickets, remainingTickets)
}
```

## Package Level Scope

Variables and functions defined outside any function can be accessed in all other files within the same package.

## Running Multiple Files

If you're using different files for the same package, you can't just run:

```bash
go run main.go  # Won't work if you have other files
```

You must provide all the files:

```bash
go run main.go helper.go
```

Or better, run all files in the current directory:

```bash
go run .
```

## Multiple Packages

Different package files should be separated in different folders:

```
project/
├── main.go           # package main
└── helper/
    └── helper.go     # package helper
```

### Exporting

To export any function or variable in Go, capitalize its first letter:

```go
// helper/helper.go
package helper

// Exported - accessible outside the package
func ValidateUserInput(firstName string, lastName string) bool {
    return len(firstName) >= 2 && len(lastName) >= 2
}

// Not exported - only accessible within the package
func internalHelper() {
    // ...
}
```

### Importing

Import packages relative to the module name:

```go
// main.go
package main

import (
    "fmt"
    "booking-app/helper"  // module-name/package-name
)

func main() {
    isValid := helper.ValidateUserInput("Shubham", "Singh")
    fmt.Println(isValid)
}
```

## Three Levels of Scope

### 1. Local Scope

Declaration within a function or block:

```go
func example() {
    localVar := "only here"  // Only accessible in this function
    
    if true {
        blockVar := "in block"  // Only accessible in this block
    }
}
```

### 2. Package Scope

Declaration outside all functions:

```go
package main

var packageVar = "everywhere in main package"

func example1() {
    fmt.Println(packageVar)  // Accessible
}

func example2() {
    fmt.Println(packageVar)  // Accessible
}
```

### 3. Global Scope

Declaration outside all functions with uppercase first letter:

```go
package helper

var GlobalVar = "everywhere across all packages"  // Exported
```

```go
package main

import "mymodule/helper"

func main() {
    fmt.Println(helper.GlobalVar)  // Accessible from another package
}
```

## Summary

| Scope | Declaration Location | First Letter | Accessible From |
|-------|---------------------|--------------|-----------------|
| Local | Inside function/block | Any | Same function/block |
| Package | Outside functions | Lowercase | Same package |
| Global | Outside functions | Uppercase | Any package |
