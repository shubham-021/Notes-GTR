# Data Types and Pointers

## Static Typing

Go is a statically typed language. You need to tell the Go compiler the data type when declaring a variable.

### Type Inference

Go can infer the type when you assign a value:

```go
var userName // Error: no value assigned, Go cannot infer the type
userName = "Shubham"
```

The correct way is to either:

```go
// Option 1: Declare with type, assign later
var userName string
userName = "Shubham"

// Option 2: Declare and assign together (type inferred)
userName := "Shubham"
```

## Pointers

Pointers hold the memory address of a value:

```go
var name = "Shubham"

fmt.Println(name)   // Prints the value: Shubham
fmt.Println(&name)  // Prints the memory address: 0xc000010230
```

The `&` operator gets the memory address of a variable.

## User Input with Pointers

When taking user input with `fmt.Scan`, you must provide a memory location:

```go
var name string
fmt.Println("Enter your first name: ")

// Wrong - passing the value
fmt.Scan(name)

// Correct - passing the memory address
fmt.Scan(&name)
```

## Practical Example

Here's a complete example showing user input and validation:

```go
package main

import (
    "fmt"
    "strings"
)

func getUserInput() (string, string, string, uint) {
    var firstName string
    var lastName string
    var emailAddress string
    var userTickets uint

    fmt.Println("Enter your first name: ")
    fmt.Scan(&firstName)

    fmt.Println("Enter your last name: ")
    fmt.Scan(&lastName)

    fmt.Println("Enter your email address: ")
    fmt.Scan(&emailAddress)

    fmt.Println("Enter number of tickets: ")
    fmt.Scan(&userTickets)

    return firstName, lastName, emailAddress, userTickets
}

func validateUserInput(firstName string, lastName string, emailAddress string, userTickets uint) (bool, bool, bool) {
    isValidName := len(firstName) >= 2 && len(lastName) >= 2
    isValidEmail := strings.Contains(emailAddress, "@")
    isValidTicketNumber := userTickets > 0

    return isValidName, isValidEmail, isValidTicketNumber
}
```

This example demonstrates:
- Multiple return values from functions
- Using pointers with `fmt.Scan`
- String validation with the `strings` package
